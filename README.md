# AdvancedServerList-OG

AdvancedServerList-OG is the TrueOG-maintained variant of AdvancedServerList for Purpur 1.19.4.

## What is different from normal AdvancedServerList?

- Uses TrueOG's live config files and branding.
- Player caching is fully removed.
- No IP addresses are stored.
- Player placeholders are not part of this OG variant.
- Build/project naming is changed to `AdvancedServerList-OG`.

## Notes

This project keeps the same general MOTD/profile system focus as AdvancedServerList, while matching TrueOG deployment and branding requirements.
